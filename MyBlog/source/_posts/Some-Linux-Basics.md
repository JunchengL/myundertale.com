# 使用SCP拷贝文件：

```shell
scp user@host:/path/to/file /path/to/put/local

# -r: 递归拷贝所有文件（文件夹内）
```

# Grep命令的使用：

```shell
grep -i #不区分大小写
grep -r #查找带有关键字内容的文本内容
grep -v #查找不带关键字的结果
grep -w #只找关键字
```
